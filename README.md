# Cute Date Invite

A playful React page that asks your girl on a date. The `Yes` button celebrates, and the `No` button runs away from the mouse, touch, keyboard focus, and clicks.

## Run It

```bash
npm install
npm run dev
```

Then open the local URL Vite prints in your terminal, usually:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

The production files will be created in `dist/`.

## Customize

Edit the wording in `src/App.jsx`. You can change the colors and layout in `src/App.css`, and replace the bouquet illustration at `public/date-flowers.svg`.
